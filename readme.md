# Building a blog with ES6 and Metalsmith

[Read the post here](http://cameronjroe.com/writing/building-a-blog-in-es6-with-metalsmith/)

### Installation

1. Clone the repo

2. Install the packages
```sh
$ npm install
```

3. Run the build
```sh
npm run build
```

4. Start a server
```sh
serve build
```
